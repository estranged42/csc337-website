Final Study Guide

  1. HTTP Request - Know the basics of an HTTP request, just the actual request, no telnet or curl parts. 
          GET /some/document.html HTTP/1.1
      host: www.example.com
  2. Know how to denote PHP code blocks: <?php /* code goes here */ ?> Know where and when you need to start/stop PHP execution in the flow of HTML.
  3. Know basic type information. Know how to get information on what type a variable holds.
  4. Understand the $_GET, $_POST, $_SERVER superglobals.
  5. HTML Forms - Know how to create an HTML form, the <form> element and the <input> element
  6. Array - Know how to create an array. Know how to access elements of an array. Know how to iterate over elements of an array. Know how to add elements to an existing array. Know the difference between automatically assigned numeric array keys and explicitly set array keys. Know how multi-dimensional arrays are constructed and accessed.
  7. Review the security concerns with user input, SQL injection, and Cross-Site Scripting attacks.
  8. Know how to properly return JSON data from a PHP program.
  9. Type coercion - Know how and when types are automatically converted in PHP. Know what PHP considers to be 'true' and 'false'. Know what happens when you add/subtract strings and numbers.
  10. Accessibility - Know what accessibility concerns to be mindful of when doing web development.
  11. Classes - Know how to create a basic class. Know how to define properties and methods. Know how to inherit from a base class. Understand visibility and scope.
  12. PDO / SQL - Know how to connect to a MySQL database using PDO. Know how to INSERT, UPDATE, and SELECT data from a MySQL database using PHP PDO objects.
  13. AJAX - Know the basic idea behind AJAX: Creating a new HTTP request behind the scenes, and registering a callback function to be executed when that request completes. Know how to use and call the ajax() helper function from assignment 6.
  14. Midterm review - Know the basics of how to construct an HTML page. How to get references to Elements in javascript, how to update properties of an element with javascript.
